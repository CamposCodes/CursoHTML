- O curso aborda os seguintes temas:

    - Aula 01: Introdução ao HTML       
    - Aula 02: Hello World              
    - Aula 03: Títulos                  
    - Aula 04: Parágrafos               
    - Aula 05: Formatação de Texto      
    - Aula 06: Links                    
    - Aula 07: Imagens                  
    - Aula 08: Listas                   
    - Aula 09: Formulários (parte 1)    
    - Aula 10: Formulários (parte 2)    
    - Aula 11: Tabelas                         
    - Aula 12: Áudio
    - Aula 13: Vídeo
    - Aula 14: Tags importantes
